# Privacy Policy for Discogs Edit Helper

**Effective Date:** 2026

Discogs Edit Helper operates entirely within the user's browser.

## Data Collection

This extension does not collect, transmit, store, or sell personal data.

## Website Access

The extension reads information from Discogs pages solely to extract track and release information for editing purposes. All processing happens locally in the browser.

## Data Storage

Any saved settings are stored locally using the browser's storage API. No data is sent to external servers.

## Third Parties

The extension does not use analytics, tracking scripts, remote code, or third-party services.
